# skillmill
A programmatic, constraint-driven worksheet generator.

**Supported Curriculum**

Singapore MOE Mathematics (Primary 1–3), arithmetic drills only:

| Level | Curriculum Node | Coverage |
| --- | --- | --- |
| P1 | Count and write to 100 | Horizontal + vertical drills |
| P1 | Add & subtract within 10 | Horizontal + vertical drills |
| P1 | Add & subtract within 20 | Horizontal + vertical drills |
| P2 | Add & subtract within 100 | Horizontal + vertical drills |
| P2 | Multiply (2, 3, 4, 5, 10 times tables) | Horizontal + vertical drills |
| P2 | Divide (2, 3, 4, 5, 10 times tables) | Horizontal + vertical drills |
| P3 | Add & subtract within 10,000 | Horizontal + vertical drills |
| P3 | Multiply (6, 7, 8, 9 times tables) | Horizontal + vertical drills |
| P3 | Divide (6, 7, 8, 9 times tables) | Horizontal + vertical drills |
